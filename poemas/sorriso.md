# SORRISO :smile:

Pra iluminar meus amigos,

Pra demonstrar meu carinho,

Pra demonstrar meu respeito

Pelo ser humano,

Para ser humano.



Para animar o meu próprio

Meu sorriso dá bom dia,

Sem se preocupar

Com nenhuma avaria

Pra poder caminhar

Com toda maestria.




Quem sabe o outro

Não use imitar?

Quem sabe o chorando

Não possa sonhar,

Pés firmes no chão

Pra continuar?

O sorriso é um abraço à distância.
